## Premise contradictions surfaced this session

Documented here so future sessions don't re-derive them:

1. **Handoff doc had wrong storage API for extension telemetry.** Said `localStorage["toneguard_telemetry"]`. Actual: `chrome.storage.local["tg_telemetry_summary"]` (extension storage API, different from DOM localStorage). PWA does use DOM localStorage at key `toneguard_telemetry_summary`.
2. **MCP `~/.toneguard/learning.json` has never been written on this machine.** The MCP server has not run with persistent storage active. The store is created on first `LearningStore.save()` — nothing locally triggers that. Empty == no MCP telemetry to read.
3. **Stale stat-cache on CLAUDE.md.** Showed `M` in `git status` with empty `git diff`. Cleared by `git update-index --refresh CLAUDE.md`. Not new behavior, just easy to spook on if not aware.
