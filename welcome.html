Add your Claude API key

ToneGuard uses Claude AI to check your messages. You'll need a free Anthropic account and an API key — most users spend $1–3 per month on usage.

  1. Go to console.anthropic.com and create an account.
  2. Open Settings → API Keys, click Create Key, and copy it.
  3. Paste it below. Your key stays in this browser — ToneGuard never sees it, and your messages go directly to Anthropic.

How it works

  1. Write your message and hit Send as usual
  2. ToneGuard checks tone and clarity in the background
  3. If the message looks good, it sends instantly
  4. If there's an issue, you'll see a suggestion with an explanation
  5. Choose to use the suggestion, edit it, or send your original
